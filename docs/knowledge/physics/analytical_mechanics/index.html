解析力学

ばねの運動

次のばねとおもりの運動を考える。

天井からつるされたばね定数 \(k\) の先に質量 \(m\) のおもりをつける。 おもりのついていないときのばねの自然長を \(l\) とし、ばね自体の質量は考えないとする。 天井から垂直下向きを \(x\) 軸の正の方向とし、おもりを初期位置 \(x = l\) 、 初速度ゼロで手を離したとする。手を離した時刻を \(t = 0\) とし、その後の単振動の 振幅と初期位相、および振動数を求めることを考える。

天井の位置を \(x = 0\) とし、時刻 \(t\) での運動しているおもりの位置を \(x(t)\) とする。 このときばねの伸びは \(x(t) - l\) であるから、運動方程式は次のようになる。

\[\begin{split}m \frac{d^2x}{dt^2} &= mg -k(x - l) \\ &= -kx + kl + mg \\ &= -k(x - l - \frac{mg}{k}) \\ m \frac{d^2}{dt^2} (x - l -\frac{mg}{k}) &= -k(x - l - \frac{mg}{k}) \\ m \ddot{X} &= -kX\end{split}\]

ここで \(X(t) = x(t) - l - \frac{mg}{k}\) とおいた。 すると

\[\ddot{X} = - \frac{k}{m} X = - \omega^2 X \quad (\omega = \sqrt{\frac{k}{m}} とした)\]

\(X(t) = \sin \omega t, X(t) = \cos \omega t\) は上式を満たすから、 この微分方程式の特解である。よって、これらの線形結合も解（一般解）であるから、

\[X(t) = A \sin \omega t + B \cos \omega t = C \cos (\omega t + \phi)\]

とおける。但し、 \(0 \leq \phi < 2 \pi\) とする。

初期位置 \(x = l\) から、 \(x(0) = l\) であるので、 \(X(0) = x(0) - l - \frac{mg}{k} = - \frac{mg}{k}\) となる。 また、初速度ゼロであるから \(\dot{x(0)} = 0\) なので、 \(\dot{X(0)} = \dot{x(0)} = 0\) より、

\[\begin{split}X(0) &= B = C \cos(\phi) = - \frac{mg}{k} \\ \dot{X(0)} &= \omega A = - \omega C \sin (\phi) = 0\end{split}\]

となる。下の式から \(A = 0\) であり、 \(\phi = 0, \pi\) である。

\(\phi = 0\) のとき、 \(\cos \phi = 1\) であるから \(X(0) = B = C = - \frac{mg}{k}\) となる。 よって、

\[\begin{split}X(t) &= - \frac{mg}{k} \cos (\omega t) \\ x(t) - l - \frac{mg}{k} &= - \frac{mg}{k} \cos (\omega t) \\ \therefore x(t) &= l + \frac{mg}{k} - \frac{mg}{k} \cos (\omega t)\end{split}\]

となる。 \(\phi = \pi\) のときは、 \(\cos \phi = -1\) であるから \(X(0) = B = -C = - \frac{mg}{k}\) となるので、

\[\begin{split}X(t) &= \frac{mg}{k} \cos (\omega t + \pi) \\ \therefore x(t) &= l + \frac{mg}{k} + \frac{mg}{k} \cos (\omega t + \pi)\end{split}\]
